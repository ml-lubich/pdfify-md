# markpdf-cli

Simple CLI wrapper for [@ml-lubich/markpdf](https://www.npmjs.com/package/@ml-lubich/markpdf).

## Installation

```bash
npm install -g markpdf-cli
```

## Usage

```bash
markpdf file.md
```

This package simply provides the `markpdf` command by depending on `@ml-lubich/markpdf`.

For full documentation, see the [main repository](https://github.com/ml-lubich/markpdf).
